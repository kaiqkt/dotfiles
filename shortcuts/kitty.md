# Kitty

Font: `Hack Nerd Font Mono 12pt` | Right `Option` key acts as `Alt`

## Tabs

| Binding        | Description          |
| -------------- | -------------------- |
| `Cmd+T`        | New tab              |
| `Cmd+W`        | Close tab            |
| `Cmd+1-9`      | Switch to tab by num |
| `Ctrl+Shift+.` | Move tab right       |
| `Ctrl+Shift+,` | Move tab left        |
| `Ctrl+Shift+Q` | Close tab            |

## Windows (splits)

| Binding          | Description        |
| ---------------- | ------------------ |
| `Ctrl+Shift+Enter` | New window       |
| `Ctrl+Shift+W`   | Close window       |
| `Ctrl+Shift+]`   | Next window        |
| `Ctrl+Shift+[`   | Previous window    |
| `Ctrl+Shift+F`   | Move window forward|
| `Ctrl+Shift+B`   | Move window back   |

## Scrollback

| Binding              | Description              |
| -------------------- | ------------------------ |
| `Ctrl+Shift+Up`      | Scroll up                |
| `Ctrl+Shift+Down`    | Scroll down              |
| `Ctrl+Shift+Page_Up` | Scroll page up           |
| `Ctrl+Shift+Page_Down` | Scroll page down       |
| `Ctrl+Shift+Home`    | Scroll to top            |
| `Ctrl+Shift+End`     | Scroll to bottom         |
| `Ctrl+Shift+H`       | Browse scrollback in pager (`less`) |

## Font size

| Binding          | Description  |
| ---------------- | ------------ |
| `Cmd++`          | Increase     |
| `Cmd+-`          | Decrease     |
| `Cmd+0`          | Reset        |

## Misc

| Binding              | Description               |
| -------------------- | ------------------------- |
| `Ctrl+Click`         | Open link under cursor    |
| `Ctrl+Shift+C`       | Copy to clipboard         |
| `Ctrl+Shift+V`       | Paste from clipboard      |
| `Ctrl+Shift+S`       | Paste from primary selection |
| `Ctrl+Shift+F2`      | Edit kitty.conf           |
| `Ctrl+Shift+F5`      | Reload config             |
| `Ctrl+Shift+Escape`  | Open kitty shell          |
